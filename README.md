# Interviews

I will be storing code here that has to do with my interviews.

## How to run
All javascript files can be ran via node
```
npm install
node javascript/*.js
```

## How to run Xavier
Xavier is a small MEAN SPA webpage

### Dependencies
1. MongoDB
2. Node & Npm
3. Nodemon
4. Git (for bower)

### Code to run to get started
```
cd Xavier
npm install
bower install
npm start
```
